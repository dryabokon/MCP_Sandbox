# calculator

## When to use
Call this tool whenever the user asks to compute, calculate, or evaluate any math expression.

## Input
- `expression` (string): a valid Python arithmetic expression, e.g. `"2 + 2"` or `"100 * 3.14"`

## Output
Returns a string with the expression and its result, e.g. `"2 + 2 = 4"`

## Notes
- Supports `+`, `-`, `*`, `/`, `**` (power), and parentheses
- Do not pass words or variables — only numeric expressions
